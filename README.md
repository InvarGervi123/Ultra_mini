# Ultra_mini

A Python-based mini game / application developed with **high‑tech engineering standards**, emphasizing clean architecture, modular design, and professional development workflow.

This repository reflects a **real-world software development process**: incremental feature delivery, focused bug fixes, and clear commit history documenting each technical improvement.

---

## Key Features

### Menu & Input Flow

* Start menu with **Enter key** activation
* ESC‑based navigation:

  * Menu → Exit
  * Menu → Back
  * End scene → Menu
* End menu reset via Enter key

### UI & Widgets System

* Fixed text widget length limitations
* Improved widget stability and edge‑case handling
* Refined interaction logic and layout behavior

### Graphics & Sprites

* Sprite system added and integrated into the rendering flow
* Clean separation between visual assets and logic

### Audio System

* Background soundtrack support
* Application‑level audio playback enabled
